HTML Tags

<a> Tag

The <a> tag is used to create hyperlinks that connect a web page to another section on the same web page, or to another web page.

<a href="destination">Target Destination</a>

HTML Form Components

Form Tag
Used to create a form that collects user input.
Label
Provides a description for an input field.
Input Field
Allows users to enter information such as text or password.
Radio Button
Allows a user to choose one option from multiple choices.
Male Female
Textarea
Allows the user to enter longer text such as comments.
Submit Button
Sends form data to the server.
Reset Button
Clears all the entered data in the form.
