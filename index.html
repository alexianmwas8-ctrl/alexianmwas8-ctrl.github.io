HTML as One of My Core Skills

What is HTML?

HTML stands for HyperText Markup Language. It is used to structure content on the web using elements and tags.

Structural Elements

Examples include html, head, body, header, main, section, and footer. They define the structure of a webpage.

Text Formatting Elements

Elements such as p, h1–h6, strong, and em are used to organize and emphasize text.

Lists

HTML supports ordered lists, unordered lists, and description lists for grouping related items.

Links and Images

The anchor tag creates links, while the image tag is used to display images.

Tables

Tables organize data into rows and columns using table, tr, th, and td.

Forms

Forms collect user input using elements such as form, input, label, and textarea.

Semantic Elements

Semantic elements like header, nav, main, section, article, and footer give meaning to the content.
